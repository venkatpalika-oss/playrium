# Country Rollout Roadmap

Tracks the suggested order for adding new countries and their experiences to Playrium, as decided in issue #16.

Goal: maximize cultural variety early, rather than clustering similar regions together.

## Suggested order

1. Korea
2. China
3. Brazil
4. Mexico
5. Egypt
6. Italy
7. Thailand
8. Vietnam
9. Turkiye
10. Greece

## How to use this list

- Treat this as a starting sequence, not a strict contract. Re-prioritize a country sooner if a particularly strong experience concept comes up for it, or if traffic/audience data suggests demand.
- When a country is added, update this file to mark it done and move the next entry to the top of near-term planning.
- Currently live countries (India, Japan, Philippines) are not listed here since they already shipped; this file only tracks the future rollout queue.

## Status

None of the countries above have been started yet. This document exists purely to preserve the prioritization decision from issue #16 so it isn't lost or re-litigated later.
